# jzoffer
java版剑指offer代码，包含剑指offer所有算法代码，统一用java语言进行了实现  
对算法感兴趣的同学可以一起学习一下，或者将要找工作面试笔试的可以参考一下  

## 算法口决：  
* 难题首选动归
* 受阻贪心暴力
* 考虑分治思想
* 配合排序哈希
